# Michel Benchimol - RM 567345

Um e-commerce especializado em produtos sustentáveis e ecológicos, desenvolvido para promover um estilo de vida mais consciente e responsável com o meio ambiente.

🚀 Objetivo
Criar uma interface responsiva utilizando HTML, CSS, Bootstrap e Grid Layout, garantindo uma experiência moderna e acessível em diferentes dispositivos.

🛒 Tipos de Produtos
Roupas e acessórios sustentáveis

📑 Estrutura do Projeto
1. Página Inicial (Home)
Header: Logo, menu de navegação e ícone de carrinho.

Banner: Carrossel com promoções e produtos em destaque.

Seções de Produtos: Grid de categorias (Roupas, Beleza, Casa, etc.) com cards do Bootstrap.

2. Página de Categorias
Lista de produtos em grid com imagem, nome, preço e botão "Adicionar ao Carrinho".

Filtros laterais (preço, tipo de produto, marca).

3. Página de Detalhes do Produto
Informações detalhadas: imagem, descrição, preço e botão de compra.

4. Página de Contato
Formulário estilizado com Bootstrap (nome, email, assunto e mensagem).

5. Footer
Links para redes sociais, informações de contato e políticas da loja.
